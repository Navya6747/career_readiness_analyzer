# 🚀 AI Career Readiness Analyzer PRO

An intelligent web application that analyzes a user's resume and suggests the best career paths based on skills, ATS score, and market trends.

---

## 📌 Features

* 🔐 Simple Login System
* 📄 Resume Upload (PDF)
* 🧠 Automatic Skill Extraction
* 📊 ATS Score Calculation
* 💼 Job Role Matching
* ❌ Skill Gap Analysis
* 🎓 Course Recommendations
* 🏆 Top 3 Career Suggestions
* 🗺 Personalized 6-Month Roadmap
* 📈 Resume Ranking Score
* 📊 Data Visualization (Graphs & Pie Chart)
* 💬 Career Chatbot
* 📥 Downloadable Report

## 📊 How It Works

1. User logs into the system
2. Uploads resume (PDF)
3. System extracts skills using keyword matching
4. Calculates ATS score for different job roles
5. Identifies missing skills
6. Recommends courses and career paths
7. Displays insights using graphs and charts
